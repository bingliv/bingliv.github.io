markdown语法

常用语法

  • *+空格 无序列表

  • 数字+.+空格 有序列表

  • 段落前后加两个* 表示加粗

  • 段落前后加一个*表示变斜

如下：

标题 #：一级标题 # # 表示二级，依次类推
代码块 一个代码块:’ ‘ 多行代码块：’’’ ‘’’
斜体加粗 要加粗的文本前三个*
删除 两个~~包围
超链接 格式：[超链接名字]（地址）
分割线 三个* 或三个-
换行 <br/>
区块 >+空格
区块嵌套 >
>>
插入图片 ![图片名称](图片地址)
对齐 用：实现，默认左对齐
转化连接 被<>包围的部分会自动转变为链接
锚点 跳转到 [ 目录 ]
