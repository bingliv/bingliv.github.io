建立网站-hexo篇

发表于2023-03-25|更新于2023-03-25
|阅读量:

[TOC]

一.环境配置

  1. 需要下载两个软件：==git==和==node.js==
    git下载教程
    node.js下载教程（后续补上）

二. 本地博客搭建

  1. 新建一个文件夹,命名为Ablog（可根据自己喜好取）
  2. 点进这个文件夹（空的），然后点击鼠标右键，点击git bash here(目的是在这个文件路径下执行下面的命令)
  3. 依次执行下面的命令
  • 安装==hexo-cli==
    npm install hexo-cli -g
  • 建立博客目录blog(执行完这一步后，Ablog文件内会出现一个blog文件夹)
    hexo init blog
  • 进入blog文件夹
    cd blog
  • 配置环境
    npm install
  • 上传网页
    hexo server或hexo s
  1. 输完以上4个命令后，会出现一个网址，复制进入浏览器即可打开网址默认面，本地博客创建完成。

三. 发布一篇新文章

  1. 进入文件夹blog，在空白处点击鼠标右键，点击git bash here
  2. 输入下面命令，即可创建一个“文章名字.md”文件。
    hexo new '文章名字'
